# Awesome

> A curated list of awesome lists — [awesome.re](https://awesome.re)

Source: [sindresorhus/awesome](https://github.com/sindresorhus/awesome) — 350k+ stars

## What It Is

The Awesome project provides curated lists of awesome resources across hundreds of categories.
Each topic area links to a separate GitHub repository maintained by the community.

## Major Categories

- **Platforms**: Node.js, iOS, Android, Electron, Linux, macOS, AWS, Firebase, Kubernetes, and more
- **Programming Languages**: JavaScript, Python, Rust, Go, Java, C++, Ruby, PHP, and 50+ others
- **Development**: Front-end frameworks (React, Vue, Angular), back-end platforms (Flask, Django, Rails), testing tools
- **Computer Science**: Machine learning, data science, cryptography, computer vision, quantum computing
- **Specialized Fields**: Gaming, databases, security, DevOps, blockchain, healthcare, education
